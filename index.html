One-time Google Setup (5 minutes)

Do this once, then paste your Client ID into index.html. No backend, no server, no cost.

  1. 1

    Go to Google Cloud Console

    Visit console.cloud.google.com → Create a new project → name it SmartEmailSorter

  2. 2

    Enable Gmail API

    APIs & Services → Library → search Gmail API → Enable

  3. 3

    Create OAuth Consent Screen

    APIs & Services → OAuth Consent Screen → External → fill App name → Add scope: gmail.readonly → Add your Gmail as test user → Save

  4. 4

    Create OAuth Client ID

    Credentials → Create Credentials → OAuth Client ID → Web application
    Authorised JavaScript origins: https://YOUR_USERNAME.github.io
    No redirect URI needed — leave it blank.

  5. 5

    Paste Client ID into index.html

    Copy the Client ID (looks like 123456789.apps.googleusercontent.com)
    Open index.html on GitHub → find this line near the top → replace YOUR_GOOGLE_CLIENT_ID_HERE:

    <meta name="google-client-id" content="PASTE_YOUR_CLIENT_ID_HERE.apps.googleusercontent.com" />
  6. ✓

    Done — Sign in with Google works!

    Push to GitHub, open your GitHub Pages URL, click Sign in with Google. Real emails will load in the Inbox tab.
